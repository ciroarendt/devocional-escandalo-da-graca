# PRP: Landing Page CloudrSync Storage — Proposta de Parceria

## Objetivo
Criar uma landing page profissional e responsiva com Tailwind CSS apresentando a proposta de parceria estratégica do CloudrSync Storage.

## Arquitetura
- Arquivo único `index.html` com Tailwind CSS via CDN
- Design responsivo mobile-first
- Paleta escura/tech com acentos em azul e verde
- Seções com scroll suave

## Checklist de Implementação

### Estrutura Base
- [ ] HTML5 boilerplate com Tailwind CDN e meta tags
- [ ] Configuração de fontes (Inter via Google Fonts)

### Seções da Página
- [ ] **Header/Nav** — Logo CloudrSync + badge "Confidencial" + nav links
- [ ] **Hero** — Título impactante, subtítulo, nome do apresentador, data
- [ ] **O Problema** — Cards com as 4 dores do mercado (preço AWS, LGPD, câmbio, egress)
- [ ] **A Solução** — Grid com os 7 diferenciais do CloudrSync com ícones
- [ ] **Comparativo** — Barra visual R$50 vs R$137 (63% mais barato)
- [ ] **Oportunidade de Parceria** — Proposta 1PB, texto explicativo
- [ ] **Tabela de Custos** — Tabela estilizada com OPEX, depreciação, custo/TB
- [ ] **Dados Técnicos** — Erasure coding 4+2, 686TB utilizáveis, custo R$6,32/TB
- [ ] **CTA** — Seção de call to action para iniciar parceria
- [ ] **Footer** — Créditos, confidencialidade, link do site

### Polish
- [ ] Responsividade mobile/tablet/desktop
- [ ] Animações sutis (fade-in com Tailwind)
- [ ] Gradientes e sombras para profundidade visual

## Validação
- [ ] Página renderiza corretamente
- [ ] Todas as seções do documento estão representadas
- [ ] Layout responsivo funcional
- [ ] Dados numéricos corretos conforme proposta original
