# Project Analysis — CloudrSync Storage Landing Page

## Objetivo
Landing page de proposta de parceria estratégica para o CloudrSync Storage — plataforma de object storage B2B brasileira.

## Tech Stack
- HTML5 semântico
- Tailwind CSS (via CDN)
- Design responsivo (mobile-first)
- Sem framework JS (página estática)

## Conteúdo da Página (seções)
1. **Hero** — CloudrSync Storage + tagline sobre parceria
2. **O Problema** — Dores do mercado brasileiro (preço AWS, LGPD, câmbio)
3. **A Solução** — Features do CloudrSync com ícones
4. **Comparativo de Preço** — R$50/TB vs R$137/TB (AWS)
5. **Oportunidade de Parceria** — Proposta de 1PB, custos operacionais
6. **Tabela de Custos** — OPEX detalhado
7. **Arquitetura Técnica** — Erasure coding, MinIO
8. **CTA** — Call to action para contato/parceria
9. **Footer** — Dados do apresentador, confidencialidade

## Decisões de Design
- Paleta: azul tech (#0ea5e9 sky) + escuro (#0f172a slate) + verde para destaque
- Estilo: corporativo, moderno, confiável (B2B)
- Animações sutis com Tailwind
- Ícones via emojis ou SVG inline (sem dependência externa)
- Layout: single page, scroll suave

## Observações
- Documento é confidencial — manter badge "Confidencial"
- Público-alvo: potencial parceiro com infraestrutura física
- Tom: profissional, dados concretos, foco em números
