# Welcome to codrsync Cloud Workspace!

## Quick Start

1. **Open terminal**: Press `Ctrl+`` or click Terminal menu
2. **Start Claude Code**: Run `claude` in terminal
3. **Create project**: Run `codrsync init`

## Useful Commands

```bash
# Claude Code (AI assistant)
claude

# codrsync commands
codrsync init          # Start new project
codrsync start         # Launch Claude with superego
codrsync status        # View project status

# Python (with UV)
uv venv                # Create virtual environment
uv pip install <pkg>   # Install packages fast
```
