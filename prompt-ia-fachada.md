# Prompt para IA de Imagem — Fachada Coffee Shop 1864

## Como usar
Copie o prompt abaixo e cole no **Midjourney**, **DALL-E 3** (ChatGPT Plus), ou **Adobe Firefly**.
Use a foto original (IMG_4827.JPG) como referência de imagem quando possível.

## Dicas por Ferramenta

| Ferramenta | Como usar |
|---|---|
| **Midjourney v6** | Use `/imagine` com o prompt + `--ar 4:3 --v 6 --style raw`. Para melhor fidelidade, use `/describe` na foto original primeiro |
| **DALL-E 3 (ChatGPT)** | Cole o prompt no ChatGPT Plus. Peça "edit this photo" com a imagem original anexada |
| **Adobe Firefly** | Use "Generative Fill" selecionando a área da calçada + a área da floreira separadamente |
| **Stable Diffusion** | Use img2img com a foto original, denoising 0.45-0.55, prompt acima |
| **Leonardo AI** | Use "Image to Image" com strength 0.5, prompt em inglês |

## Especificação Técnica dos Materiais

### Cimento Queimado Brilhante
- **Acabamento:** Cimento queimado com aplicação de resina epóxi cristal ou verniz poliuretano brilhante
- **Cor:** Cinza Antracite / Grafite escuro (referência: Portobello Cimento Grafite)
- **Brilho:** Alto brilho (>70 GU no glossmeter)
- **Aplicação:** Sobre contrapiso nivelado, com 3 demãos de cimento queimado + 2 demãos de resina

### Floreira
- **Dimensões:** 0,80m altura x 0,40m profundidade x comprimento total da fachada de vidro
- **Material:** Concreto moldado in loco ou blocos de concreto com revestimento liso
- **Acabamento externo:** Massa corrida + pintura acrílica cinza escuro (referência: Suvinil Cinza Chumbo)
- **Impermeabilização interna:** Manta asfáltica + dreno de fundo
- **Plantas sugeridas:**
  - Jiboia (Epipremnum aureum) — pendente
  - Samambaia (Nephrolepis) — volume
  - Aspargo-pluma (Asparagus densiflorus) — textura
  - Pingo-de-ouro (Duranta erecta) — cor
  - Bromélias (Neoregelia) — destaque
  - Suculentas variadas — complemento
