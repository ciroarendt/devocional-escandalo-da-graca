# PRP — Proposta Online EPIC x Mentores

## Objetivo
Criar uma proposta comercial online (HTML single-page) com branding EPIC para apresentar ao Daniel Graciola o projeto de Workshop de Alinhamento Comercial + Cultura de Atendimento.

## Arquitetura
- Arquivo único: `index.html` (HTML + CSS + JS inline)
- Sem dependências externas além de Google Fonts
- Mobile-first, responsivo
- Paleta: tons escuros (preto/grafite) + accent laranja/âmbar (identidade EPIC work)
- Tom: executivo, direto, sem floreios
